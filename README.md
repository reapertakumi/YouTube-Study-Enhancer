# YouTube-Study-Enhancer

A lightweight Chrome extension to block Shorts, hide video suggestions, and disable the 2x speed feature for a focused, distraction-free YouTube experience. Perfect for students, researchers, and anyone who wants to watch intentionally.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎬 **Hide Shorts** | Completely removes YouTube Shorts from homepage, search results, and sidebar |
| 📺 **Hide Suggested Videos** | Removes the "Up next" sidebar while watching videos |
| ⚡ **Block 2x Speed** | Disables the hold-to-speed-up feature (spacebar or mouse click) |
| 🎨 **Modern Dark UI** | Clean, minimal popup with smooth toggle switches |
| 💾 **Persistent Settings** | Your preferences are saved automatically across sessions |

## 🎯 Why Use This?

- **For Students**: Watch lectures without getting distracted by Shorts or suggested videos
- **For Researchers**: Stay focused on the content you actually need
- **For Intentional Viewers**: No more accidentally speeding through videos and retaining nothing
